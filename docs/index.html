Installation

Deploy in 5 Steps

From zero to a fully deployed career portal. No DevOps experience required.

1

Scaffold Your Project

Run npx create-open-hire and follow the interactive prompts for your company name, website URL, brand color, and email.

2

Install Dependencies

Navigate into your new project and run npm install to grab all required packages.

3

Initialize Convex Backend

Run npx convex dev to authenticate and deploy your real-time serverless database. Leave this terminal open.

4

Create Your Admin

In a new terminal, run npx convex run seed:createDefaultAdmin to generate your dashboard login credentials.

5

Launch

Run npm run dev and visit localhost:5173. Your admin portal is at /admin.html.

Architecture

How the Ecosystem Works

A serverless, decoupled architecture built for speed and infinite scalability.

🌐

Career Portal

Static HTML/JS served via Vite. Lists active jobs, company benefits, and handles SEO metadata.

📝

Application Engine

Dynamic job pages with custom forms, AI autofill, phone inputs, multi-select fields, and file uploads.

🛡️

Admin Dashboard

Password-protected panel to create jobs, design forms, review applicants, and view analytics.

⚙️

Convex Backend

Real-time serverless database powering jobs, applications, analytics, and admin auth.

📦

CLI SDK

Interactive Node.js CLI that scaffolds, brands, and configures the entire platform from scratch.

🚀

Deploy Anywhere

Push to Vercel or Netlify with a single git push. Production-ready out of the box.
